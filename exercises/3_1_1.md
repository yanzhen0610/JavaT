### 基本架構

#### 步驟
1. 建立Main類別
    ```java
    public class Main {
    }
    ```

2. 建立main方法
    ```java
    public class Main {
        public static void main(String[] args) {
        }
    }
    ```

3. 將預運行的代碼寫在方法中
    ```java
    public class Main {
        public static void main(String[] args) {
            System.out.println("Hello, world!");
        }
    }
    ```
